## Installation

Linux:

```ascii
$ chmod +x install.sh
# ./install.sh
```

or

```ascii
# make install
```

Windows:

```ascii
bundle install
ruby main.rb
```

## Uninstall

Linux:

```ascii
# rm /usr/local/bin/g4t
```

or

```ascii
# make uninstall
```
